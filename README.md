# Rail shooter game idea
## Description
A fast-paced (arcade) rail shooter featuring 2D graphics in a 3D dungeon-themed environment, 
where the player automatically moves along a linear path while focusing on aiming and shooting with precision. 
Along the way, you must react to constant waves of enemies and various hazards, putting your reflexes and real-time responsiveness to the test. 
Progress is driven by intense action sequences designed to keep the tension high, with the goal of surviving as long as possible.

## Features
- Main gameplay mechanic: aim and shoot
- Game genre: rail shooter.
- Game mode: Singleplayer.
- Aesthetics: **dungeon-themed.
